Projeto 1 : Calculo Trabalhista

O Projeto foi passado para farermos um site que realizasse cálculos trabalhistas em que o funcionário recebe salário calculado por valor hora, bastando que o usuário informe: Valor da hora, Quantidade de Hora-trabalhada, Vale-transporte e outras deduções ,para fazer esse site utilizei do HTML para fazer espaços onde seria inserido as informações ,o CSS foi usado para deixar o site mais bonito ou seja fazer a cara do site e por fim o JAVA SCRIPT foi utilizado para fazer os cálculos e exibir seus resultados

Projeto 2 : Calculadora

O Projeto foi passado para fazermos uma calculadora foi utilizado o HTML e o CSS para fazer o rosto do programa e o JAVA SCRIPT foi utilizado para fazermos as contas e dentre outras funções de uma calculadora.
